## 注释要求

```cpp
//本题的逻辑结构：线性表/队列/栈/二叉树/图 等等
//本题的存储结构：顺序/链式 等等
//解题思路和算法：输入n个点存储在s[N]中，然后依次输入 a, b
//               然后遍历s[n]，如果a <= s[i] <= b 那么ans++,最后返回的ans就是落点数量
//效率：  时间复杂度：O(n^2)
//        空间复杂度：O(n)
/*测试数据：
输入：
5 2
1 3 7 9 11
4 6
7 12
-------------------------

输出：
0
3

*/
```
